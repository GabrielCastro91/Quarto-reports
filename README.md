# Análisis Exploratorio y Segmentación de Clientes Mediante Clustering

Éste proyecto tiene como objetivo resolver la problemática de que no existe ningún tipo de segmentación en lo que respecta a los clientes directos del negocio. Existen dos canales de venta denominados "Reseller" e "Internet". Para el primero de ellos existen variables que funcionan como un tipo de segmentación, pero no existe variable alguna para el segundo canal que comprende al cliente común/persona física.

Une vez procesados y analizados los datos, se utilizó la técnica de segmentación **RFM (Recency, Frequency and Monetary Value)** y se complementó este acercamiento al problema mediante la implementación de un modelo de clustering **K-Means**, lo que permitió segmentar la base de clientes en 4 utilizando su historial de compras.
